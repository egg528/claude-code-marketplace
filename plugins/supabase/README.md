# Supabase Plugin

Supabase database development integration for Claude Code.

## Features

- PostgreSQL database management
- Row Level Security (RLS) policies
- Authentication integration
- Storage management

## Prerequisites

Set the `SUPABASE_ACCESS_TOKEN` environment variable:

```bash
export SUPABASE_ACCESS_TOKEN="your-token-here"
```

Get your access token from [Supabase Dashboard](https://supabase.com/dashboard/account/tokens).

## Installation

```bash
/plugin install supabase@egg528-marketplace
```
